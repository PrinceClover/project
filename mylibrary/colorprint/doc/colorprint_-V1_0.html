模板概述
  • 本模板库封装了ASCII格式控制处理"\033[m"系列字符，能为控制台输出添加简单的颜色及控制效果
使用环境
  1. 只能用于Linux/Unix的C++标准输出流(cout)输出
  2. 可以输出自定义类型，但需要使用者自行重载 << 运算符
注意事项
  • 使用此模板库时需将源文件复制到使用此库的目录，并包含"colorprint.hpp"
快速入门
  • 库中只有一个模板类 ColorPrint：
    • tmpleate<typename PrintType> class ColorPrint ;
  • 所有的工作都由 ColorPrint 对象来完成，一个简单的示例如下：
    1. #include <iostream>                  //标准IO流库
    2. #include "colorprint.hpp"           //打印模板库
    3. using std::cout;                         
    4. using std::endl;

    6. int main(void)
    7. {
    8.    colorprint::ColorPrint<int> cp;                        //声明一个打印int型的ColorPrint对象
    9.    cp.setFormat(colorprint::BG_RED);                   //设置背景颜色为红色
    10.    cp.appendFormat(colorprint::FONT_BLUE);      //添加格式-设置字体颜色为蓝色
    11.    cp.appendFormat(colorprint::UNDERLINE);      //添加格式-设置下划线
    12.    int testint = 1024;                                               //待打印变量
    13.    cout << cp.printOne(testint) << endl;               //打印变量testint
    14.    //打印一串变量
    15.    cout << cp.openFormat() << "hello " << testint  << cp.closeFormat() << endl; 
    16.    return 0;
    17. }

类型/常量定义
  • 格式标志位/格式常量定义
    • 原型：typedef unsigned int format_flag_t
    • 描述：
      1. 其每一位代表一种格式，其中低0~7位表示字体颜色，8~15位表示背景颜色，0~15位表示颜色格式(Color)，16~20位表示控制格式(Control)
      2. 构造格式标志位时使用或(I)运算，如红色背景绿色字体加粗表示为 (colorprint::BG_RED | colorprint::FONT_GREEN | colorprint::HIGHLIGHT)，使用appendFormat()方法追加格式时，若设定了颜色格式(Color)，则会覆盖原有的颜色格式(Color)，控制格式(Control)不会被覆盖
      3. 反色会使隐藏效果失效，隐藏效果会使字体隐藏，并隐藏所有效果，除了背景颜色
