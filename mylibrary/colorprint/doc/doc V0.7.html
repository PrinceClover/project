author@PrinceClover(871353611@qq.com)  v0.7

模板概述
  • 本模板库封装了ASCII格式控制处理"\033[m"系列字符，能为控制台输出添加简单的颜色及控制效果
使用环境
  1. 只能用于Linux/Unix的C++标准输出流(cout)输出
  2. 可以输出自定义类型，但需要使用者自行重载 << 运算符
注意事项
  • 使用此模板库时需将源文件复制到使用此库的目录，并包含"colorprint.cc"
  • 使用命名空间Prince_CP或Prince_CP::ColorPrint
快速入门
  • 库中只有一个模板类 ColorPrint：
    • tmpleate<typename PrintType> class ColorPrint ;
  • 所有的工作都由 ColorPrint 对象来完成，一个简单的示例如下：
    1. #include "colorprint.cc"           //打印模板库
    2. #include <iostream>              //标准IO流库
    3. using std::cout;                         
    4. using std::endl;
    5. using Prince_CP::ColorPrint;     //使用打印模板库的命名空间

    7. int main(void)
    8. {
    9.    ColorPrint<int> cp;                        //声明一个打印int型的ColorPrint对象
    10.    cp.setFormat(BG_RED);                   //设置背景颜色为红色
    11.    cp.appendFormat(FONT_BLUE);      //添加格式-设置字体颜色为蓝色
    12.    cp.appendFormat(UNDERLINE);      //添加格式-设置下划线
    13.    int testint = 1024;                           //待打印变量
    14.    cout << cp.print(testint) << endl;    //打印变量testint
    15.    //打印一串变量
    16.    cout << cp.openFormat() << "hello " << testint  << cp.closeFormat() << endl; 
    17.    return 0;
    18. }

类型/宏定义
  • 格式标志位/格式宏定义
    • 原型：typedef unsigned int formatflag
    • 描述：
      1. 其每一位代表一种格式，其中低0~7位表示字体颜色，8~15位表示背景颜色，0~15位表示颜色格式(Color)，16~20位表示控制格式(Control)
      2. 构造格式标志位时使用或(I)运算，如红色背景绿色字体加粗表示为 (BG_RED | FONT_GREEN | HIGHLIGHT)，使用appendFormat()方法追加格式时，若设定了颜色格式(Color)，则会覆盖原有的颜色格式(Color)，控制格式(Control)不会被覆盖
      3. 反色会使隐藏效果失效，隐藏效果会使字体隐藏，并隐藏所有效果，除了背景颜色
